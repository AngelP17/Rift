# Rift Glossary

- **Decision ID**: Hash of transaction + features + prediction; receipt for replay
- **Calibrated score**: Probability adjusted so 0.9 means ~90% empirically
- **Conformal band**: high_confidence_fraud | review_needed | high_confidence_legit
- **ECE**: Expected Calibration Error
- **PR-AUC**: Precision-Recall Area Under Curve
- **Recall@1% FPR**: Recall when false positive rate is 1%
- **Replay**: Re-run stored inputs through model to verify same outcome

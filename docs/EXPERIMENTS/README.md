# Rift Experiments

This folder contains Jupyter notebooks and HTML exports for experiments:

- **01_relational_vs_tabular** – Compare XGBoost tabular vs GraphSAGE vs hybrid
- **02_calibration_and_conformal** – ECE, Brier, reliability curves
- **03_temporal_leakage** – Random vs chronological vs rolling splits
- **04_conformal_coverage** – Empirical coverage, average set size

See [../experiments.md](../experiments.md) for experiment design and claims.
